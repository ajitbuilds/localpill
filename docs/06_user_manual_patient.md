# LocalPill — Patient User Manual

> **Kiske liye**: Medicine search karne wale patients
> **Kya hai LocalPill**: Ek app jisse aap apne nearby pharmacies se seedha puch sakte hain ki unke paas koi medicine available hai ya nahi.

---

## App kaise Open Karein

**Web Browser se (Mobile ya Desktop)**:
→ `https://localpill-upcharmitra.web.app` open karein

**App Install Karna (Recommended)**:
- 📱 **Android**: Chrome mein site open karein → "Add to Home Screen" banner aayega → Install karein
- 🍎 **iPhone/iPad**: Safari mein site open karein → Share button (↑) → "Add to Home Screen"

---

## Step 1: Login Karna

1. App open hone par **"Get Started"** button dabayein
2. Apna **10-digit mobile number** enter karein (bina +91 ke)
3. **"Send OTP"** button dabayein
4. Aapke phone par **6-digit OTP** aayega SMS se
5. OTP boxes mein code enter karein
6. App automatically aapko login kar dega ✅

> 💡 **Tip**: OTP copy karke boxes mein paste bhi kar sakte hain — automatically fill ho jaata hai!

---

## Step 2: Pehli Baar — Naam Batayein

Pehli baar login karne par:
1. "**I am a Patient**" select karein
2. Apna **poora naam** enter karein
3. **"Continue"** dabayein

---

## Step 3: Medicine Search Karna

### Dashboard se
- Home screen par **"🔍 Search Medicine"** button dabayein

### What you'll see on the Search screen:

**Box 1 — Medicines ka Naam**
- Medicine ka naam type karein aur Enter ya comma dabayein
- Multiple medicines add kar sakte hain
- Galat medicine → ❌ button se hata sakte hain

**Box 2 — Prescription Upload (Optional)**
- Agar doctor ne prescription di hai → **"Upload Prescription"** dabayein
- Camera se photo le sakte hain ya gallery se select karein
- Image crop kar sakte hain properly

**Location (Automatic)**
- App automatically aapka location detect karega
- Pehli baar permission maangega → **"Allow"** karein

3. Sab fill karne ke baad → **"Find Nearby Pharmacies"** button dabayein

---

## Step 4: Results Dekhna

Request bhejne ke baad **Results Screen** par aayenge:

### Status Timeline (Top par)
- 📡 **Request Sent** — Aapki request bheji gayi
- 🔔 **Pharmacies Notified** — Nearby pharmacies ko notification gayi
- 💬 **Responses Received** — Pharmacies ne respond kiya
- 🤝 **Matched** — Aapne ek pharmacy choose ki

### Pharmacy Cards (Neche)
Jo pharmacies available hain, unki detail dikhegi:
- Pharmacy ka naam
- Kitni door hai (distance)
- **"Chat"** button — seedha baat karo

### Map View
- **"Show Map"** button → Nearby pharmacies map par dikhenge
- Green dots = Online pharmacies
- Marker dabao → Pharmacy ki detail dekhein

---

## Step 5: Pharmacy se Chat Karna

1. Pharmacy card par **"Chat"** button dabayein
2. Chat screen khulegi
3. Type karein aur **Send** dabayein
4. Pharmacy real-time mein jawab degi

**Chat mein kya pooch sakte hain**:
- Medicine ki price
- Kitni stock hai
- Koi substitute available hai?
- Address ya directions

**Prescription Share Karna (Agar pharmacy maange)**:
- Pharmacy "Request Prescription" bhejegi
- Aap chat mein "Share" button se prescription share kar sakte hain
- **Secure link** milti hai — sirf 10 minutes ke liye valid

**Request Complete Karna**:
- Jab pharmacy mil jaaye → **"Mark as Complete"** button dabayein
- Request band ho jaayegi

---

## Step 6: Request History Dekhna

- Menu → **"My History"** (ya dashboard par history section)
- Saari purani requests dikhegi
- Status dekho: Pending 🟡, Matched 🟢, Closed ⚫

---

## Notifications

**Push Notifications Allow Karein** (Browser pop-up aane par) — tabhi batayenge jab:
- Koi pharmacy ne respond kiya
- Chat mein nayi message aayi

**Bell Icon (🔔)** Header mein:
- Unread notifications ka count dikhta hai
- Click karo → Notification list
- Notification click karo → Direct us screen par jaayein

---

## Zaruri Baatein (Important)

> ⚠️ **LocalPill ek enquiry platform hai** — yeh medicine order ya deliver nahi karta. Pharmacy sirf availability batati hai — medicine khud jaake leni hogi.

> ℹ️ **Medical Advice nahi**: LocalPill se koi doctor se advice nahi milti. Medicines doctor ke prescription ke anusar hi lein.

> 🔒 **Privacy**: Aapki prescription sirf aap aur responding pharmacy dekh sakti hai.

---

## Troubleshooting

| Problem | Solution |
|---------|---------|
| OTP nahi aaya | 2-3 minute ruko, phir try karo |
| Location detect nahi ho raha | Browser settings mein location allow karo |
| Map load nahi ho raha | Internet connection check karo |
| Koi pharmacy nahi dikh rahi | Radius badaya ja sakta hai (Settings mein) |
| Notification nahi aa rahi | Browser mein notification permission check karo |
